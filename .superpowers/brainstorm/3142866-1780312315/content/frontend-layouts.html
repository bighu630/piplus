Option C: Layered Workbench

Mobile direction

Recommended mobile behavior

  • 左侧树在手机上变成全屏 drawer
  • 对话区保持单栏，输入框固定底部
  • session 详情 / tool calls 用底部 sheet 打开

Avoid

  • 手机上保留三栏或小侧栏
  • 大量卡片包卡片的后台式布局
  • 把所有状态都常驻显示，压缩对话空间

Current recommendation

我建议以 Option C 为基础，视觉上做成更精致的 conversation-first studio：桌面端看起来接近 A，但把多余信息收进抽屉；手机端也最自然。
